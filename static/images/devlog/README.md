# Devlog Screenshots

This directory stores screenshots for playground development logs.

## Naming Convention

Screenshots should be named descriptively with the project and date:

- `tank-2025-12-07-3d.png` - Fish tank 3D conversion
- `tank-2025-12-07-2d.png` - Fish tank 2D version
- `[project]-[YYYY-MM-DD]-[description].png`

## Adding Screenshots

1. Take a screenshot of the current state
2. Save it here with the naming convention above
3. Reference it in the devlog entry in `/src/lib/data/playground.ts`
4. The screenshot will be displayed in the devlog section

## Recommended Size

- Max width: 1200px
- Optimize images before committing
